# AI-Powered Cloud Reliability & Auto-Healing Platform

## Overview

A hands-on DevOps project that demonstrates cloud application deployment, monitoring, observability, and AI-assisted incident analysis. The project is being built step by step using industry-standard DevOps tools and practices.

## Current Features

- ✅ Flask REST API
- ✅ Health Check Endpoint (`/health`)
- ✅ Error Simulation Endpoint (`/simulate-error`)
- ✅ Prometheus Metrics Endpoint (`/metrics`)
- ✅ Unit Testing with Pytest
- ✅ Git & GitHub Version Control

## Tech Stack

- Python
- Flask
- Pytest
- Prometheus Client
- Git
- GitHub

## Run Locally

```bash
pip install -r application/requirements.txt
python application/app.py
```

Run tests:

```bash
pytest application/tests -v
```
